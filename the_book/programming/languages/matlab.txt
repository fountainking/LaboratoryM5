**MATLAB**

MATLAB is a proprietary multi-paradigm
programming language and numeric
computing environment developed by
MathWorks. MATLAB allows matrix
manipulations, plotting of functions
and data, implementation of algorithms,
creation of user interfaces, and
interfacing with programs written in
other languages.
