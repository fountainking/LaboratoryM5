C Sharp (programming language)

C# is a general-purpose high-level
programming language supporting
multiple paradigms. C# encompasses
static typing, strong typing, lexically
scoped, imperative, declarative,
functional, generic, object-oriented
(class-based), and component-oriented
programming disciplines.
