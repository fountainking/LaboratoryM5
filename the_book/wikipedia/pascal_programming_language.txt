Pascal (programming language)

Pascal is an imperative and procedural
programming language, designed by
Niklaus Wirth as a small, efficient
language intended to encourage good
programming practices using structured
programming and data structuring. It is
named after French mathematician,
philosopher and physicist Blaise
Pascal.
