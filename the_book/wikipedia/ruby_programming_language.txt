Ruby (programming language)

Ruby is a general-purpose programming
language. It was designed with an
emphasis on programming productivity
and simplicity. In Ruby, everything is
an object, including primitive data
types. It was developed in the
mid-1990s by Yukihiro "Matz" Matsumoto
in Japan.
