TypeScript

TypeScript (TS) is a high-level
programming language that adds static
typing with optional type annotations
to JavaScript. It is designed for
developing large applications and
transpiles to JavaScript. It is
developed by Microsoft as free and
open-source software released under an
Apache License 2.0.
