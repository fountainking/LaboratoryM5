Create fake WiFi networks with custom SSIDs.

üîß TROUBLESHOOTING

SD Card Not Detected

  • Ensure card is formatted as FAT32
  • Try reinserting the card
  • Check card capacity (max 32GB for FAT32)
  • Verify card is not corrupted

WiFi Connection Issues

  • Check if network requires captive portal login
  • Verify password is correct
  • Some enterprise networks are not supported
  • Try disconnecting and reconnecting

Audio Playback Problems

  • Check if sound is enabled in settings
  • Verify MP3 bitrate is compatible (128-320 kbps)
  • Ensure file is not corrupted
  • Reduce radio buffer if streaming stutters
